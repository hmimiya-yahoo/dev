## 仕様変更

`mdxx-ssg new:page` が `pages/*.tsx` を生成しないように

かわりに、 `pages/[slug].tsx` が共通の記事要素になりました。

## new feature: tags support

frontmatter の yaml を使って、タグを宣言できるようになりました

```yaml
---
title: my-title
created: 1589127481769
tags: [a, b, c]
---
# hello
```

トップページにタグ一覧が表示されます。

`/tags/[tag]` ページで、そのページに紐付いている記事一覧を出せるようにしました。
