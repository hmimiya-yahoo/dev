# 第一章 Server(less) Mode

```ts
// pages/api/hello.ts

export default (req, res) => {
  res.json({ text: "hello" });
};
```

`pages/api/*` は next.js によって `/api/hello` のルーティングにマッピングされます。

```ts
const res = await fetch("/api/hello");
const data = await res.json();
console.log(data); // => {text: "hello"}
```
